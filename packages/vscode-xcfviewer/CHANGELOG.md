# Change Log

All notable changes to the "vscode-xcfviewer" extension will be documented in this file.

## [Unreleased]

## [0.1.0] - 2026-02-11

### Added

- Initial release
- XCF file preview with custom editor
- Hierarchical layer tree view in Explorer sidebar
- Toggle individual layer visibility with eye icon
- Show all layers command
- Hide all layers command
- Refresh layers command
- Real-time image updates when toggling layers
- Support for layer groups
- Integration with xcfreader library v1.0.1

### Features

- Automatic file association for .xcf files
- Visual indicators for layer visibility (eye icons)
- Crisp image rendering with optimized display
- Dark theme compatible UI
- Retain context when switching tabs

### Known Issues

- Large files (>50MB) may take time to load
- Layer effects not yet rendered
- Text layers appear rasterized
- No undo/redo support for layer visibility changes
